My Project & My Training

Website portofolio ini saya buat secara mandiri tanpa menggunakan framework, melainkan hanya dengan HTML, CSS, dan JavaScript murni. Proyek ini saya kerjakan sebagai bentuk latihan sekaligus media untuk menunjukkan kemampuan dasar saya dalam membangun tampilan web dari nol, mulai dari struktur, desain, hingga interaksi sederhana di dalamnya.

Pembuatan website ini membantu saya lebih memahami konsep dasar pengembangan web, serta melatih konsistensi dan kerapihan dalam penulisan kode. Website ini juga menjadi langkah awal saya sebagai calon Full-Stack Web Developer yang terus belajar dan berkembang untuk menghadapi tantangan di dunia teknologi.

Salah satu proyek yang saya buat adalah sebuah website untuk mengelola data kelas, yang dibangun menggunakan framework Laravel dan terhubung langsung dengan database MySQL. Website ini dirancang untuk menampilkan seluruh isi tabel data kelas secara dinamis, sehingga memudahkan dalam pengelolaan informasi.

Pada website ini terdapat tiga fungsi utama, yaitu menambahkan data, mengedit data, dan menghapus data. Proyek ini saya kerjakan sebagai latihan membangun aplikasi berbasis web dengan fitur CRUD (Create, Read, Update, Delete), sekaligus memperdalam pemahaman saya dalam menggunakan Laravel sebagai backend framework modern yang powerful.

Saya juga terlibat dalam proyek pembuatan game berjudul Lost History menggunakan game engine Godot. Dalam proyek ini, saya bertanggung jawab sebagai programmer/scripter yang mengatur logika permainan, interaksi karakter, serta implementasi sistem teka-teki dan pergerakan musuh.

Game ini mengusung konsep petualangan dalam alam bawah sadar, di mana pemain harus memecahkan teka-teki untuk keluar dari dunia mimpi. Saya mengimplementasikan mekanik dasar seperti gerakan karakter, musuh (seperti Slime dan Skeleton), serta sistem level dan dialog. Proyek ini memberi saya banyak pengalaman dalam mengelola alur logika game dan berkolaborasi dalam tim pengembangan.

Saya telah menyelesaikan pelatihan Database Foundations yang diselenggarakan oleh Oracle Academy pada tanggal 28 Desember 2024. Dalam pelatihan ini, saya mempelajari dasar-dasar pengelolaan database seperti konsep relasional, desain tabel, SQL dasar, serta prinsip-prinsip normalisasi.

Pelatihan ini memperkuat pemahaman saya dalam mengelola basis data, yang sangat membantu dalam pengembangan aplikasi berbasis Laravel yang terhubung langsung dengan MySQL. Sertifikat ini menjadi salah satu bukti komitmen saya untuk terus belajar dan berkembang di bidang teknologi informasi.

Pada tanggal 30-31 Juli 2024, saya menyelesaikan pelatihan Data Analytics untuk Siswa SMA/Sederajat melalui program Digital Talent Scholarship 2024 yang diselenggarakan oleh Kementerian Kominfo dan Puslitbang SDP3I. Pelatihan ini berlangsung selama 18 jam dan membekali saya dengan dasar-dasar analisis data, pengolahan data menggunakan spreadsheet, serta visualisasi data untuk pengambilan keputusan.

Pelatihan ini melengkapi pemahaman saya dalam mengelola data secara efisien, yang sangat berguna untuk pengembangan aplikasi berbasis database dan pelaporan data secara visual.
