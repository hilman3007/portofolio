Hai, Saya Hilman Nugraha


Saya adalah siswa SMK NEGERI 4 BANDUNG jurusan Rekayasa Perangkat Lunak (RPL). Sejak duduk di bangku SMP, saya sudah memiliki ketertarikan yang besar terhadap dunia Informatika, terutama dalam bidang pemrograman Website. Tujuan saya adalah menjadi seorang Full-Stack Web Developer yang menguasai pengembangan aplikasi dari sisi frontend maupun backend.


Saat ini saya belajar dengan materi sekolah dan secara mandiri melalui berbagai sumber online seperti dokumentasi resmi, video tutorial, dan artikel. Saya mempelajari bahasa dan teknologi seperti HTML, CSS, JavaScript, PHP, serta framework Laravel. Saya juga mulai terbiasa menggunakan MySQL dalam pengembangan proyek.


Meskipun belum pernah mengikuti bootcamp, atau kegiatan organisasi, saya aktif mengasah kemampuan saya dengan mengerjakan proyek-proyek pribadi seperti website portofolio, sistem login, dan aplikasi manajemen data sederhana. Dengan konsistensi dan semangat belajar, saya percaya diri dapat terus berkembang hingga menjadi developer yang profesional dan siap menghadapi tantangan dunia industri teknologi.
